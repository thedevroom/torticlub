# 1. CREATIVE DIRECTION — ¿Quién es TortiClub?

## 1.1 Esencia (del brand book)

> **No hacemos muchas tortillas. Hacemos la tortilla perfecta para que la disfrutes a tu manera.**
> **Bienvenido al club.**

## 1.2 ADN

| Es | No es |
|----|--------|
| Marca de producto | Restaurante / bar / franquicia genérica |
| Club de pertenencia | Delivery sin alma |
| Compartir como ritual | Gastronomía pretenciosa |
| Cercano, divertido, directo, honesto | Lujo frío / meme forzado / startup postureo |
| Barcelona (Sant Andreu) con ambición de marca | Solo “tortillería de barrio” |

## 1.3 Mantra inmutable

```
Parte. Comparte. Repite.
```

Aparece en: logo lockup, pegatinas, packaging, tarjetas, poster, stories, bio IG, web mockups.  
**Nunca se explica. Nunca se alarga. Nunca se reescribe.**

## 1.4 Personalidad

Si TortiClub fuera una persona:

- Elegante sin ser frío
- Seguro, tranquilo, con mucho gusto
- No grita; por eso se le escucha
- Habla como un amigo: sin rodeos, con humor, sin postureo
- Cercano · Divertido · Directo · Honesto

## 1.5 Emoción de marca

Pertenencia > hambre.  
El producto es la tortilla; la promesa es **formar parte del Club**.

# 3. IDENTIDAD VISUAL

## 3.1 Logotipo

### Lockup principal (`logo-1`, brand book, packaging)

```
torticlub ⬤eyes ®
parte. comparte. repite.
```

- Wordmark: **todo minúsculas**, una sola palabra, sin espacio
- Peso: bold / black, sans geométrica redondeada
- Kerning: ajustado, compacto, moderno
- Símbolo: a la derecha del wordmark, escala ~ altura de las minúsculas altas
- ® en superíndice junto al símbolo
- Tagline: centrada bajo el lockup, peso regular/medium, tamaño ~30–40% del wordmark

### Símbolo oficial (`logo-oficial.png`)

- Círculo amarillo sólido
- Dos ojos blancos ovoides que se tocan en el centro (forma de “8” horizontal / compartir)
- Pupilas negras circulares, centradas o ligeramente expresivas
- **Sin boca** (salvo expresiones de mascota en brand book)
- Reconocible sin el nombre

### Variantes observadas

| Variante | Uso |
|----------|-----|
| Símbolo solo (ojos amarillos) | App icon, favicon, preloader, merch |
| Lockup horizontal + tagline | Web header, packaging, print |
| Wordmark + ojos sin tagline | Poster top, stories, emails |
| Ojos lineales negros (solo contorno) | Dorso tarjeta, minimal |
| Wordmark uppercase “TORTICLUB” | Solo en pieza de highlights IG (excepción editorial) |

## 3.2 Mascota — Los Ojos

Del brand book: **8 expresiones**

| Estado | Lectura |
|--------|---------|
| normal | neutro, observa |
| feliz | mirada alegre / semi-cerrada |
| sorprendido | pupila grande / “O.O” |
| dudoso | mirada lateral |
| cansado | párpados bajos |
| enamorado | corazones en pupilas |
| mischief | ceño / malicia amable |
| curioso | mirada inclinada / “¿?” |

### Reglas de uso

- Los ojos son **personaje**, no emoji decorativo
- Tienen función: observan, reaccionan, acompañan
- Pueden recortarse a media cara (poster, tarjeta) como firma de composición
- Nunca distraen del producto
- En motion: siguen el cursor de forma sutil; morph a símbolo en preloader

## 3.3 Paleta de color (fuente brand book)

| Token | Hex brand book | Rol | Notas de muestreo en assets |
|-------|----------------|-----|------------------------------|
| `surface` / cream | `#F7F3E8` | Fondo principal | Muestreado ~`#F8F0E7` en logo-1 |
| `primary` / yellow | `#FFD23F` | Acento, mascota, CTAs | Logo oficial ~`#FCBF09`; usar token book |
| `ink` / black | `#111111` | Texto, iconos, contraste | Pupilas ~`#161615` |
| white | `#FFFFFF` | Ojos, highlights | — |

### Reglas de color

- **Máximo 3 colores estructurales** (crema, amarillo, negro)
- El amarillo nunca es “fondo de sección entero” salvo mascota/hero gráfico
- Fondos de piezas: siempre crema cálido, no blanco clínico
- Iconos secundarios: círculo crema suave o stroke negro
- Separadores: línea negra fina + **punto amarillo** en el centro (firma visual recurrente)

### Firma gráfica recurrente: el punto amarillo

Aparece en:

- Separadores de sección (`— ● —`)
- Entre palabras del mantra en tipografía circular
- Como “period” de color en “parte. comparte. repite.”
- Bajo títulos en stories

## 3.4 Tipografía

| Rol | Familia (brand book) | Uso |
|-----|----------------------|-----|
| Primary | **PP Neue Montreal** Medium / Bold | Todo el sistema |
| Display | Bold / Black, tracking tight | Headlines “PARTE. COMPARTE.” |
| Body | Medium, tracking neutral | Párrafos, UI |
| Caption / Label | Medium, uppercase o sentence | Icon labels, meta |
| Accent script | Manuscrita casual (solo asides) | “¡Nos vemos pronto!” |

### Jerarquía observada

1. **Display XL** — 1–3 palabras, mayúsculas, peso máximo, leading muy cerrado  
2. **Title** — sentence case bold (“gracias. eres parte del club.”)  
3. **Overline / category** — “TORTILLAS PARA COMPARTIR.” (amarillo o negro)  
4. **Body** — 2–4 líneas máximo por bloque  
5. **Label** — icon captions en mayúsculas pequeñas  
6. **Script accent** — una sola línea emocional  

### Ortografía de marca

- Lockup: `torticlub` minúsculas
- En prosa: `TortiClub` o `Torticlub` (assets mezclan; **canónico recomendado: TortiClub** en prosa, `torticlub` en logo)
- Mantra: siempre con puntos: `Parte. Comparte. Repite.` o minúsculas `parte. comparte. repite.`

## 3.6 Grid, espaciado, composición

### Principios

- **Muchísimo aire negativo** (crema como protagonista)
- Composición asimétrica elegante (texto izq / mascota der en poster y agradecimiento)
- Alineación fuerte a ejes verticales
- Márgenes generosos tipo editorial (no “template Bootstrap”)
- Elementos táctiles: sombra suave bajo tarjetas y packaging (foto realista, no UI shadow dura)

### Firmas compositivas

1. **Mascota a media cara** entrando desde el borde derecho  
2. **Mantra en display apilado** con puntos  
3. **Separador línea + punto amarillo**  
4. **Producto fotográfico anclado abajo-centro** con sombra natural  
5. **Badge circular** “HECHO CON AMOR PARA TI”  
6. **Footer de 3 columnas**: QR + redes + “Bienvenido al Club”  

### Border radius

- Círculos perfectos para mascota, stickers, highlights  
- Cards / packaging: radios suaves (MD–LG), nunca squircle exagerado  
- Botones: full pill o radio generoso  

### Sombras

- Muy suaves, cálidas (no negras puras)
- Largas y difusas bajo producto y mockups
- En UI: casi ausentes; el contraste lo da el crema vs negro

# 13. ARQUITECTURA DE LA EXPERIENCIA DIGITAL

## 13.1 Principio

No es una web de restaurante.  
Es el **lanzamiento de un producto de marca** (Apple / Nothing / Teenage Engineering) aplicado a tortilla-para-compartir.

## 13.2 Mapa de experiencia (home = película)

```
[PRELOADER]
  ojos → “bienvenido al club.” → morph logo → telón

[ESCENA 1 — IDENTIDAD]
  lockup + mantra + CTA “Entrar al Club”
  muchísimo aire, cero foto de tortilla todavía

[ESCENA 2 — REVELACIÓN]
  scroll: la tortilla emerge como objeto de diseño

[ESCENA 3 — TRANSFORMACIÓN / PARTE]
  tortilla fija; se divide 1 → 2 → 4
  aparecen SOLO · DUO · CLUB como conceptos → luego interactivos

[ESCENA 4 — DESEO / SABORES]
  interacción: hover/click revela sabores e ingredientes
  sin lista de carta

[ESCENA 5 — CONFIGURADOR]
  Formato → Sabores → Cantidad → Entrega → Resumen
  fluidez tipo configurador premium (no menú)

[ESCENA 6 — PERTENENCIA]
  caja se abre / “El Club te está esperando”
  CTA pedido

[ESCENA 7 — CLUB / HISTORIA / LOGÍSTICA]
  Sant Andreu, recogida, envíos, valores
  (contenido de stories elevado a escena)

[NAV INVISIBLE]
  icono → Club · Carta/Producto · Historia · Pedir
```

## 13.3 Arquitectura de información

```
/                     Experiencia principal (storytelling + pedido)
/pedir                Configurador (deep link)
/club                 Manifiesto / pertenencia
/historia             Origen Sant Andreu + valores
/ formalos            SOLO · DUO · CLUB (opcional deep)
```

Mobile-first. Desktop = misma narrativa con más aire y stage de producto mayor.

- Motion (Framer Motion moderno) · Lenis
